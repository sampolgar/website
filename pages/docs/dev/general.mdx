## js function reminder

### arrow 1

- `function` and const functions are almost the same.
- `async function greeter(greeting, name) {}` - same as
- `const greeter = async (greeting, name) => {}` - same as
- `async (greeting, name) => {}`

### arrow 2

- preferred:`describe("hello", function() {})`
- not preferred: `describe("hello", () => {})`

## function export + destructure

```js
async function deploy(hre) {
  const namedAccounts = hre.getNamedAccounts,
    deployments = hre.deployments;
}
module.exports.default = deploy();
```

same as

```js
module.exports = async (hre) => {
  const { getNamedAccounts, deployments } = hre;
};
```

same as

```js
module.exports = async ({ getNamedAccounts, deployments }) => {};
```
